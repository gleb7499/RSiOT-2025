# RSIOTLab1
# Руководство по запуску проекта на Docker

Это руководство поможет вам запустить проект на Docker, используя предоставленные файлы конфигурации и код.

## 1. Подготовка окружения

Перед началом убедитесь, что у вас установлены Docker и Docker Compose. Вы можете скачать их с официального сайта [Docker].

## 2. Структура файлов

Убедитесь, что у вас есть следующая структура файлов:
/project-directory
├── docker-compose.yml
├── Dockerfile
├── go.mod
└── main.go

- **docker-compose.yml** - файл конфигурации для Docker Compose.
- **Dockerfile** - файл для сборки образа вашего приложения.
- **go.mod** - файл для управления зависимостями Go проекта.
- **main.go** - основной файл вашего приложения.

## 3. Запуск проекта

1. Перейдите в каталог проекта:

   cd /path/to/your/project-directory
   
2. Запустите Docker Compose:
   
   docker-compose up --build
   
3. Откройте браузер и перейдите по адресу [http://localhost:8052] для доступа к вашему приложению.
